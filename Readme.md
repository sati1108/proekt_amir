# GitHub User Finder

**Автор: Афашагов Амир Асланович

## Описание
Приложение на Python с графическим интерфейсом (Tkinter), которое позволяет находить информацию о пользователях GitHub и сохранять их в локальный список избранного.

## Как использовать API
Программа использует публичное **GitHub REST API**. 
- Базовый URL: `https://github.com{username}`
- Регистрация (токен) не требуется для небольшого количества запросов (до 60 в час).

## Примеры использования
1. Введите в поле поиска `mojombo`.
2. Нажмите "Найти" — в списке отобразятся данные профиля.
3. Нажмите "В избранное", чтобы сохранить пользователя.
4. Нажмите "Показать избранное", чтобы увидеть сохраненных в JSON пользователей.

**Тест:** Если оставить поле пустым и нажать "Найти", программа выдаст предупреждение.
